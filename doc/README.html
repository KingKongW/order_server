• 订单数据结构
字段名 描述 类型 是否不必 备注
orderNumber 订单号 long yes key
status 订单状态 number yes 1（待确定）、2（已确定）、3（已审核）、4（已取消）
createTime 下单时间 date yes --
auditedTime 审核时间 date no --
confirmTime 确认时间 date no --
cancelTime 取消时间 date no --
source 订单来源 number yes 1（携程）、2（飞猪）、3（马蜂窝）、4（线下同行）
quantity 订单数量 number yes --
eleTicketQuantity 电子票数量 number yes 默认0
entityTicketQuantity 实体票数量 number yes 默认0
externalOrderNumber 外部订单号 string yes --
orderBelong 订单归属人 string yes --
orderBelongId 订单归属人id string yes --
goodsName 订单商品名称 string yes --
goodsCode 订单商品编码 string no --
supplier 供应商 string yes --
incomingNumber 进货单号 string yes --
incomingTime 进货时间 Date yes --
salesVolume 销售额 float yes 浮点数
unitPrice 销售单价 float yes --
rmbCost 人民币进货价 float yes --
costCurrency 进货币种 String yes --
currencyExchangeRate 进货货币汇率 float yes --
deliveryQuantity 快递件数 number yes 默认0
deliveryCost 快递费用 float yes 默认0
deliveryNumber 快递单号 string no 多个快递单号用-隔开（如：123231-312312）表示两个单号
commission 佣金 float yes 默认0
discount 优惠 float yes 默认0
profit 利润 float yes --
profitRate 利润占比 float yes --
netProfit 净利润 float yes --
netProfitRate 净利润占比 float yes --
